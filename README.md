### Hi there 👋

很多朋友都好奇“释慧利”这个ID的来历，其实这是我奶奶的名字。我不太愿意多谈这个话题，因为其中有太多遗憾。

巧合的是，17号也是我《深入理解分布式共识算法》交稿的日子。我希望这个ID能成为她存在过的印记。
